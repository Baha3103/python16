ssh-keygen - это нужно для генерации ssh ключа
.ssh - это скрытая директория ssh настроек
id_rsa.pub - в ней хранится открытый ключ

GIT - система контроля версий
git init - инициализация 
.git - создается скрытая папка

git add <file_name> <.> - для проверки статуса
git commit -m 'comments' - он фирсирует файл и создает хэш сумму
git push origin master - отправка всех локальных коммитов в удаленный репозиторий

git remote add origin git@github.com:Baha3103/python16.git

git clone - просто клон проекта из удаленного сервера Github

git config --global user.name 'Baha3103'
